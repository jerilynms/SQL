- Fortune 500 Analysis: Combining all of the below 
- Healthcare Billing: AS, ROUND(), CASE, GROUP BY, HAVING
- Bank Products Database: AND/OR, IN(), LIKE, Wildcards 
- Superstore Database: ORDER BY, WHERE, Aggregations
